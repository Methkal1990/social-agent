# Social-Agent: Project Overview

## Project Name
**social-agent** - A CLI/TUI-based AI-powered social media management agent for X (Twitter)

## Vision
An intelligent, autonomous social media manager that handles personal brand management on X with configurable AI autonomy, content generation from multiple sources, and comprehensive engagement automation.

## Target User
- Personal brand managers
- Individuals building their presence on X
- Content creators wanting AI assistance

## Core Value Proposition
1. **AI-First Content Creation**: Generate original content, repurpose existing material, and capitalize on trends
2. **Configurable Autonomy**: From fully autonomous to human-in-the-loop, per task
3. **Full Engagement Automation**: AI handles replies, mentions, and engagement in your voice
4. **Network Building**: Intelligent suggestions for follows and strategic engagement
5. **Learning System**: Continuously improves based on performance data

## Scope

### In Scope (v1)
- X (Twitter) platform only
- Single account management
- Single persona/voice
- Content generation (original + repurposing + trends)
- Queue-based scheduling with AI-optimized timing
- Full engagement automation
- Network building and discovery
- Basic analytics with AI learning loop
- TUI dashboard interface
- RSS feed integration for content research
- AI image generation
- Thread support
- Quote tweet strategy
- A/B testing with auto-optimization
- Confidence-based approval workflow
- Full content moderation
- Multi-format export

### Out of Scope (v1)
- Other social media platforms (future consideration)
- Multi-account management
- Multi-client/agency features
- Campaign management
- Import from existing tweet history
- Reply/thread tracking

## Key Differentiators
1. **Terminal-Native**: Full TUI experience with dashboard, not just CLI commands
2. **Deep AI Integration**: OpenRouter for model flexibility, full reasoning transparency
3. **Learning System**: Agent improves over time from performance data
4. **Configurable Everything**: Autonomy levels, content types, posting frequency, etc.
5. **No Attribution**: Posts appear completely organic

## Success Metrics
- Consistent posting without manual intervention
- Improved engagement rates over time (AI learning)
- Reduced time spent on social media management
- Growth in followers and engagement

## Development Approach
**Feature-Complete v1**: Build all planned features before first release, ensuring a polished initial experience.
